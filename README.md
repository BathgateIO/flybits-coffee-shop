# flybits-coffee-shop

> A simple point-of-sale (POS) application for a coffee shop like Starbucks



## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# clean the production build
npm run clean
```
